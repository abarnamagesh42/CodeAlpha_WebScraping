CodeAlpha Web Scraping

 Project Title:-
Job Market Analysis Using Web Scraping

 Objective:-
The objective of this project is to collect job market data from a website using Python web scraping techniques. The scraped data is cleaned, analyzed, and visualized to identify hiring trends, popular locations, and frequently advertised job roles.

Tools Used:-
- Python
- Requests
- BeautifulSoup
- Pandas
- Matplotlib
- Seaborn
- Google Colab

Website Used:-
Real Python Fake Jobs Website

https://realpython.github.io/fake-jobs/

 Data Collected:-
- Job Title
- Company Name
- Job Location

 Project Workflow:-
1. Connect to the website.
2. Scrape job details.
3. Store data in a Pandas DataFrame.
4. Save data as CSV.
5. Analyze the collected data.
6. Create visualizations.
7. Draw insights.

Visualizations:-
- Top Hiring Locations
- Top Companies Hiring
- Most Common Job Titles
- Job Distribution by Location

Key Insights:-
-Some locations have more job opportunities than others.
- A few companies post multiple vacancies.
- Certain job roles appear more frequently.
- Web scraping makes data collection faster and more efficient.

## Conclusion
This project demonstrates how Python and BeautifulSoup can be used to collect data from websites. The scraped data was analyzed and visualized to understand hiring patterns and support data-driven insights.
